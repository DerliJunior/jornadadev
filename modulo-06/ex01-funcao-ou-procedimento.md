# Parte A — Funções (modularização) 🧱

Exercício 1 — Função ou procedimento? (conceitual)
Para cada item abaixo, responda se é uma função (retorna um valor) ou um procedimento (só
executa uma ação) e justifique em uma frase:

* Str() — converte número em string
* QOut() — exibe texto na tela
* Date() — retorna a data atual
* Len() — retorna o tamanho de uma string ou array
* Upper() — retorna a string em maiúsculas
* ClearScreen() — limpa a tela

📝 Entrega: responda no campo de texto da atividade no Classroom.

---
Str() — converte número em string -> é uma function, pois converte e retorna uma string

QOut() — exibe texto na tela -> é uma procedure, pois não retorna nada

Date() — retorna a data atual -> é uma function, pois retorna uma string com a data atual

Len() — retorna o tamanho de uma string ou array -> functio, pois retorna um inteiro INT

Upper() — retorna a string em maiúsculas ->  é uma function, pois converte uma string para maiuscula e depois retorna uma nova string já formatada

ClearScreen() — limpa a tela -> é uma procedure, pois não tem retorno
